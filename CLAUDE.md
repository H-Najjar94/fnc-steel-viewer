# CLAUDE.md — session bootstrap

At the start of EVERY session, read these four files before doing anything else.
They hold the saved context, understanding, history, and working rules for this
project. Keep them updated as instructed in `04_INSTRUCTIONS.md`.

1. [01_INFO.md](01_INFO.md) — what this project is, layout, key numbers
2. [02_UNDERSTANDING.md](02_UNDERSTANDING.md) — my working mental model
3. [03_UPDATES_HISTORY.md](03_UPDATES_HISTORY.md) — append-only log of work done
4. [04_INSTRUCTIONS.md](04_INSTRUCTIONS.md) — how to work here + end-of-session rules
5. [05_CURRENT_STATE.md](05_CURRENT_STATE.md) — what's in progress / next / blocked
6. [06_DECISIONS_AND_PREFERENCES.md](06_DECISIONS_AND_PREFERENCES.md) — choices + how you want me to work

This is a structural-steel fabrication data package (drawings, CNC files, BOM/MTO),
not a software codebase. Treat all `.dwg/.nc1/.dxf/.pdf/.xlsb` as real fabrication
data: never delete/rename/overwrite without explicit confirmation.
